Задача: Создать систему учета заказов в интернет-магазине.

Описание задачи в стиле сериала "Офис":

В интернет-магазине "Dunder Mifflin" требуется вести учет заказов и товаров. Для этого необходимо создать систему учета заказов в интернет-магазине.


Классы:
Класс "Order" - представляет собой заказ, сделанный покупателем в интернет-магазине. У этого класса есть следующие свойства:
номер заказа (тип int)
дата заказа (тип Date)
статус заказа (тип String)
список товаров (тип List<Product>)

Класс "Product" - представляет собой товар, который можно купить в интернет-магазине. У этого класса есть следующие свойства:
название (тип String)
описание (тип String)
цена (тип double)
количество (тип int)

Класс "OrderTracker" - представляет собой систему учета заказов в интернет-магазине. У этого класса есть следующие методы:
addOrder(Order order) - добавляет новый заказ в систему учета
removeOrder(Order order) - удаляет заказ из системы учета
findOrderById(int orderId) - ищет заказ в системе учета по номеру заказа и возвращает найденный заказ
findOrdersByStatus(String status) - ищет заказы в системе учета по статусу и возвращает список найденных заказов


Примеры:
Создайте объект системы учета заказов в интернет-магазине.
Создайте несколько объектов заказов и товаров и добавьте их в систему учета.
Используя метод findOrderById, найдите заказ по номеру заказа и выведите его данные на экран.
Используя метод findOrdersByStatus, найдите все заказы со статусом "в обработке" и выведите их данные на экран.
Используя метод removeOrder, удалите заказ из системы учета.
